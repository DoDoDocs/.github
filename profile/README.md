# Dododocs
자동으로 **README를 생성**해주고, **코드를 문서화**해주며, 이를 바탕으로 한 **지능형 챗봇을 제공**하는 웹 기반 서비스입니다.

<br><br>

## 📍 차별성
- **최신 AI 모델 활용**: 코드 구조와 로직은 물론 개발자의 의도까지 분석하여 실질적인 인사이트를 제공합니다.

- **RAG 기반의 지능형 챗봇**: 코드 분석 결과, 원문, 이전 대화 기록을 종합적으로 고려하여 정확하고 관련성 높은 답변을 제공합니다. 

- **효율적인 토큰 관리 및 분석 기술 활용**: MSA같은 대규모 프로젝트에서도 안정적이고 빠르게 작동합니다.
<br><br>
